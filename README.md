# E-Commerce Customer Behavior Analysis Dashboard

## Overview

This project presents an interactive Power BI dashboard designed to analyze customer behavior, purchasing patterns, and business performance in an e-commerce environment. The dashboard provides insights into customer engagement, conversion performance, visitor behavior, purchase trends, and operational metrics to support data-driven decision-making.

## Objective

The primary objective of this project is to:

* Analyze customer purchasing behavior.
* Evaluate conversion performance across different visitor segments.
* Identify trends in purchases and user engagement.
* Examine the impact of traffic sources and operating systems on business performance.
* Generate actionable insights to improve customer retention and conversions.

## Tools Used

* Power BI
* Power Query
* Microsoft Excel

## Dashboard Pages

### 1. Business Snapshot

Provides a high-level overview of business performance through key metrics such as:

* Total Purchases
* Conversion Rate
* Average Page Value
* Purchase Distribution by Region
* Revenue Trends
* Visitor Type Analysis

### 2. Behavior Analysis

Focuses on understanding customer engagement and behavior through:

* Average Session Duration by Visitor Type
* Product Page Engagement Analysis
* Bounce Rate Analysis
* Conversion Rate by Visitor Segment
* Weekday vs Weekend Purchase Analysis

### 3. Usage Insights

Examines user activity and platform performance using:

* Purchase Trends by Month
* Session Trends by Month
* Operating System Performance
* Visitor Engagement Metrics
* Traffic Source Performance

## Key Insights

* Returning visitors accounted for the majority of purchases, highlighting the importance of customer retention.
* Conversion rates differed across visitor segments, indicating varying purchasing behaviors.
* Purchase activity showed noticeable monthly trends and fluctuations.
* Engagement metrics revealed that returning visitors demonstrated higher interaction levels compared to other visitor groups.
* Traffic sources and operating systems displayed different conversion and purchase performance patterns.

## Dataset

The dataset contains e-commerce website session and purchase information, including visitor type, session duration, page engagement metrics, traffic sources, operating systems, conversion indicators, and purchase-related data.


## Conclusion

This project demonstrates how Power BI can be used to transform raw e-commerce data into meaningful business insights. By analyzing customer behavior, engagement patterns, and conversion metrics, the dashboard helps identify opportunities to improve user experience, increase conversions, and support strategic business decisions.
